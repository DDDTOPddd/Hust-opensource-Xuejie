1.  **纠错与建议:** 如果你发现任何错误、笔误，或者有更好的理解和学习资源，请大胆地提出 `Issue`。
2.  **交流讨论:** 针对特定课程或知识点，欢迎在相关的 `Issue` 区进行开放和友好的讨论。
3.  **内容补充 (Pull Requests):** 如果你有该课程相关的、并且认为有价值补充的公开资料，可以考虑发起 `Pull Request`。
